Right now, our "Add to Cart" button just adds a quantity of 1. We'll fix this by adding a quantity selector (+ / - buttons) to the ProductDetailScreen. This will allow a user to add 5, 10, or any number of an item to their cart at once.

lib/providers/cart_provider.dart - Part 2: The addItem Function
We will modify the addItem function to accept a new quantity argument.

 // ... (inside CartProvider class)

  // 1. THIS IS THE OLD FUNCTION:
  // void addItem(String id, String name, double price) { ... }

  // 2. THIS IS THE NEW, UPDATED FUNCTION:
  void addItem(String id, String name, double price, int quantity) {
    // 3. Check if the item is already in the cart
    var index = _items.indexWhere((item) => item.id == id);

    if (index != -1) {
      // 4. If YES: Add the new quantity to the existing quantity
      _items[index].quantity += quantity;
    } else {
      // 5. If NO: Add the item with the specified quantity
      _items.add(CartItem(
        id: id,
        name: name,
        price: price,
        quantity: quantity, // Use the quantity from the parameter
      ));
    }

    _saveCart(); // This is the same
    notifyListeners(); // This is the same
  }

  // ... (removeItem and all other functions are unchanged)

(Part A):
●	void addItem(..., int quantity): We added the new quantity parameter.
●	_items[index].quantity += quantity;: This is the key logic. If the user adds 5, and they already have 2 in the cart, their new total will be 7.
●	CartItem(..., quantity: quantity): If it's a new item, we create it using the quantity the user selected (e.g., 5).

Let's add the functions to change the quantity. Add these inside your _ProductDetailScreenState class.

 // ... (inside _ProductDetailScreenState)

  // 1. ADD THIS FUNCTION
  void _incrementQuantity() {
    setState(() {
      _quantity++;
    });
  }

  // 2. ADD THIS FUNCTION
  void _decrementQuantity() {
    // We don't want to go below 1
    if (_quantity > 1) {
      setState(() {
        _quantity--;
      });
    }
  }

(Part B, Step 2):
●	_incrementQuantity(): A simple function that adds 1 to _quantity and calls setState to rebuild the UI.
●	_decrementQuantity(): Does the same, but first checks that _quantity is greater than 1 so we can't have 0 or negative items.
